# project-bnb-reports

Questa repository contiene documentazioni, codice sorgente e reports del progetto.

Le tecnologie utilizzate sono:

- **Frontend**: vue.js, tailwindCSS
- **Backend**: Laravel, MySQL
- **API**: RESTful
